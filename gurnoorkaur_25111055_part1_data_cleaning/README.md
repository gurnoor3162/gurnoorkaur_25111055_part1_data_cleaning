# Sales Performance Dashboard – Part 1

## Project Overview

This project focuses on cleaning and analyzing a sales dataset using Microsoft Excel.

## Tasks Completed

- Cleaned missing values
- Filled missing Region with "Unknown"
- Filled missing Ship Mode with "Unknown"
- Replaced missing Discount values with 0
- Flagged invalid dates
- Flagged duplicate Order IDs
- Flagged high discounts (>50%)
- Created calculated sales, profit, and profit margin columns
- Built PivotTables for analysis
- Created an Excel dashboard

## Dashboard

The dashboard includes:

- Sales by Category
- Sales by Region
- Monthly Sales Trend
- Sales by Ship Mode
- Profit by Category
- Orders by Status

## Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Excel Formulas
